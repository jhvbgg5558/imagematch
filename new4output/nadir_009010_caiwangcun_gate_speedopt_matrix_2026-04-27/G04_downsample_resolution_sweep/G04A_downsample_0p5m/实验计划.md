### 问题与结论
| 问题/观察 | 结论 |
| --- | --- |
| 0.5m/pix 跑通但没有加速 RoMa | G04A RoMa rerank=3720.388s，反而高于 G02 的 1400.535s。 |
| 精度明显破坏 | Layer-2 从 G02 2.072707m 上升到 6.039019m；Layer-3 从 0.498889m 上升到 441.898506m。 |
| PnP 候选全部可解不代表定位可靠 | G04A PnP 状态为 ok:100，但 tiepoint 地面误差 RMSE 已达 441.898506m，说明匹配/几何对应质量被降采样破坏。 |
| 结论 | 0.5m/pix 在本 gate 中不作为可接受加速方案；后续如继续研究降分辨率，应先检查 query 内参缩放、candidate affine、RoMa 输入尺寸策略和验证 tiepoint 对齐链路。 |
